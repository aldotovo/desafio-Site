As tags HTML são como comandos especiais que usamos para dizer ao navegador como mostrar o conteúdo de uma página da web. Elas funcionam como etiquetas que marcam o início e o fim de diferentes tipos de conteúdo, como títulos, parágrafos, imagens e links.

Principais tags

(voltar)
  • html: Define o documento HTML, é o início do código. Ex:strong
  • head: É o cabeçalho da página, contendo metadados. São informações que não aparecem no corpo do site.
  • title: É o título da página.
  • body: Esta relacionada ao conteúdo da página, o corpo do site.
  • h1 a h6: Títulos e subtítulos.
    • h1:

      Representa o título principal de uma página.

    • h2:

      Subtítulo.

    • h3:

      Subtítulo.

    • h4:

      Subtítulo.

    • h5:
      Subtítulo.
    • h6:
      Subtítulo. Esta tag é a menor da hierarquia h.
  • mark: é usada para destacar ou marcar texto, geralmente representando informações importantes ou em evidência. O texto envolvido por essa tag é exibido com um fundo amarelo por padrão, simulando um marcador de texto físico. Ex: mark
  • small: reduz o tamanho da fonte do texto contido, frequentemente usada para notas de rodapé, comentários, informações secundárias ou disclaimers. Ela ajuda a hierarquizar informações e diminuir o impacto visual de trechos menos importantes. Ex: small
  • i: aplica um estilo itálico ao texto, geralmente usada para enfatizar palavras, indicar termos estrangeiros, ou para títulos de livros e publicações científicas. Apesar de ter um efeito visual semelhante ao itálico. Ex: itálico
  • u: sublinha o texto, tradicionalmente utilizada para destacar informações ou enfatizar links. No entanto, o uso de sublinhado em texto comum pode gerar confusão com links, por isso é menos comum em design moderno. Ex: sublinhado
  • strong: é usada para dar forte ênfase ao texto, geralmente aplicando negrito, indicando que o conteúdo tem alta relevância no contexto. strong
  • a: cria um link para outros documentos, páginas ou recursos. É uma das tags mais importantes da web, permitindo navegação e conectividade entre sites. Pode conter atributos como href (para o destino do link) e target (para controlar como ele é aberto).
  • hr: insere uma linha horizontal para separar seções de conteúdo. Geralmente é usada como divisor visual.
  • sub: exibe texto em formato subscrito, ligeiramente abaixo da linha base do texto principal. É comum em fórmulas químicas (ex.: H₂O) ou anotações científicas.
  • sup: apresenta texto em formato sobrescrito, ligeiramente acima da linha base. É muito usada em expoentes matemáticos ou referências bibliográficas (ex.:1).
  • blockquote: é usada para inserir citações em bloco, geralmente de maior comprimento. O texto é exibido com indentação ou estilização especial para diferenciá-lo do conteúdo principal.

Listas

(voltar)
  • Listas: Ordenas e Não ordenadas
    • ul: cria uma lista ordenada, em que os itens aparecem numerados ou em sequência alfabética. Ela é ideal para passos sequenciais, tutoriais ou qualquer conteúdo onde a ordem importa.
    • ol: gera uma lista não ordenada, onde os itens são exibidos com marcadores (como pontos ou símbolos). É útil para listar informações sem hierarquia ou sequência específica.
    • li: define um item dentro de uma lista ordenada ou não ordenada.

Novas tags sugeridas

(voltar)
  • font: era usada para estilizar textos, como alterar cor, tamanho ou tipo de fonte. Embora funcional, ela foi substituída por CSS para separar melhor o conteúdo da apresentação.
  • del: indica texto que foi removido ou não é mais relevante, geralmente aparecendo tachado. É útil para mostrar revisões ou alterações em conteúdo.
  • p: é usada para criar parágrafos, sendo uma das estruturas básicas de texto no HTML. Cada parágrafo é renderizado com espaçamento automático entre blocos, facilitando a leitura.
  • abbr: define uma abreviação ou acrônimo. Ao passar o mouse sobre o texto, geralmente é exibida uma descrição expandida, usando o atributo "title". É útil para melhorar acessibilidade e entendimento.
